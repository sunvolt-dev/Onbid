# 온비드 부동산 물건목록 조회서비스
**SVC-API-001**

차세대 국가자산처분시스템(온비드) Open API 활용가이드

---

## 1. 서비스 개요

### 1.1 기본 정보

| 항목 | 내용 |
|------|------|
| 서비스ID | SVC-API-001 |
| 서비스명(국문) | 온비드 부동산 물건목록 조회서비스 |
| 서비스명(영문) | OnbidRlstListSrvc |

### 1.2 서비스 설명

한국자산관리공사에서 운영하는 온비드(Onbid)에 등록된 공매 물건목록 중 물건유형이 부동산으로 분류되며 현재 입찰 중이거나 입찰예정인 부동산 공매물건 목록을 조회할 수 있는 서비스입니다.

서비스를 활용하여 전국 단위의 부동산 공매 정보를 한 번에 수집할 수 있으며 입찰 진행 상태(입찰 중 / 입찰 예정)를 기준으로 투자 시점 판단이 가능하고 물건 유형, 지역, 입찰일정 중심의 데이터 기반 분석 및 서비스 개발이 용이합니다.
