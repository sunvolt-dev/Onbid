# 온비드 물건상세 입찰정보 조회서비스

## 1. 서비스 개요

| 분류 | 항목 | 내용 |
|------|------|------|
| 서비스 정보 | 서비스ID | SVC-API-007 |
| | 서비스명(국문) | 온비드 물건상세 입찰정보 조회서비스 |
| | 서비스명(영문) | OnbidCltrBidDtlSrvc |
| | 서비스설명 | 한국자산관리공사에서 운영하는 온비드(Onbid)에 등록된 공매 물건에 대한 입찰방법 및 입찰제한정보, 회차별 입찰정보 등의 입찰정보를 상세 조회할 수 있는 서비스입니다. 조회 시 물건관리번호와 공매조건번호를 요청 시 필수값으로 입력해야 합니다. |
| 서비스 보안 | 서비스인증/권한 | [O] 서비스 Key |
| | 메시지레벨암호화 | [O] 없음 |
| | 전송레벨암호화 | [O] 없음 |
| 적용기술 수준 | 인터페이스표준 | [O] REST (GET, POST, PUT, DELETE) |
| | 교환데이터표준 | [O] XML, [O] JSON |
| 서비스URL | 운영환경 | http://apis.data.go.kr/B010003/OnbidCltrBidDtlSrvc |
| 서비스 배포정보 | 서비스버전 | 1.0 |
| | 메시지교환유형 | [O] Request-Response |
| | 메시지로깅수준 | 성공: Header / 실패: Header, Body |
